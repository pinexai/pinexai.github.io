What we're building

Every tool ships as a pip/npm install. No accounts, no API keys, no cloud dependency.

🔥
tokenspy

cProfile for LLMs — find which function is burning your AI budget. Cost flame graphs, structured tracing, evaluations, prompt versioning, and a live dashboard. All local.

python llm-observability cost-profiling openai anthropic
v0.2.0 pip install tokenspy →
🧠
agent-memory

Production-ready persistent memory for AI agents. Three-tier architecture (working, episodic, semantic) that mirrors human memory. Works with LangChain, CrewAI, AutoGen, and raw SDKs — in 3 lines.

python ai-agents memory mcp langchain crewai
v0.1.2 pip install agentcortex →
🛡️
sentrix

Red-team, eval, and monitor your LLMs — pure Python, zero config. Auto-generate adversarial tests, fingerprint vulnerabilities across models with an attack heatmap, and export OWASP / NIST compliance reports. Find vulnerabilities before your users do.

python llm-security red-teaming owasp adversarial
v0.2.0 pip install sentrix →
🗣️
dissent

15 AI executives debate any idea live — CEO, CTO, CFO, CISO, Legal, the Client, and 9 more. They argue, deadlock, and converge on a real verdict. Bring your own API key. No backend, no signup, no install — runs in your browser.

browser app ai-debate groq anthropic bring-your-own-key
v1.0.0 open dissent.html →

How we build

Every PinexAI project follows these rules. No exceptions.

🏠

Local-first

Your data stays on your machine. No telemetry, no cloud sync, no phone-home. Works offline, always.

📦

Zero config

One install command. One import. Working in seconds. We hate boilerplate as much as you do.

🔓

MIT licensed

Use it in production, fork it, sell it, we don't care. Open source means open source.

Build with us

Issues, PRs, and ideas welcome. Every project is built in the open.
